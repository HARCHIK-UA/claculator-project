\# Простий калькулятор

Консольний калькулятор для виконання базових арифметичних операцій.  

Навчальний проєкт для вивчення Git, GitHub та Python.



\## Опис

Цей проєкт представляє собою простий калькулятор з текстовим інтерфейсом, який дозволяє виконувати основні математичні операції в інтерактивному режимі.



\## Функціональність

\- Додавання двох чисел

\- Віднімання двох чисел

\- Множення двох чисел

\- Ділення двох чисел (з перевіркою ділення на нуль)

\- Піднесення до степеня

\- Обчислення остачі від ділення (модуль)



\## Вимоги

\- Python 3.7 або вище  

\- Операційна система: Windows, macOS, Linux
