# Простий калькулятор
Консольний калькулятор для базових арифметичних операцій.  
Навчальний проєкт для вивчення Git, GitHub та Python.

## Опис
Цей проєкт — простий калькулятор із текстовим інтерфейсом.  
Дозволяє виконувати основні математичні операції в інтерактивному режимі.

## Функціональність
- Додавання двох чисел
- Віднімання
- Множення
- Ділення (з перевіркою ділення на нуль)
- Піднесення до степеня
- Обчислення остачі від ділення (модуль)

## Вимоги
- Python 3.7 або вище  
- ОС: Windows / macOS / Linux

## Встановлення
1. Клонуйте репозиторій:
```bash
git clone https://github.com/HARCHIK-UA/claculator-project.git
``````
2. Перейдіть у папку проєкту:
``````bash
cd calculator-project
``````
3. Запустіть програму:
``````bash
python calculator.py
``````
## Використання

1. Оберіть операцію (+, -, *, /, **, %)

2. Введіть перше число

3. Введіть друге число

4. Отримайте результат

### Приклад роботи
=== Простий калькулятор ===
Операції: +, -, *, /, **, %
Для виходу введіть ‘exit’

Введіть операцію: -
Перше число: 10
Друге число: 6
Результат: 4.0

Введіть операцію: **
Перше число: 3
Друге число: 5
Результат: 243.0
